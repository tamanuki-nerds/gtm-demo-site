# GA4レポートの見方（基礎3画面）

> 所要時間：約20分

広告代理店の実務でよく確認する3つの画面に絞って解説します。

---

## 1. リアルタイムレポート

**使い道：** 「今この瞬間のユーザー行動」を確認する。GTMの設定が正しく動いているか確認するのに最適。

**場所：** GA4 → レポート → リアルタイム

### 見るべき箇所

| 項目 | 説明 |
|------|------|
| 過去30分のユーザー数 | 現在サイトにいるユーザー数 |
| イベント別ユーザー数 | どのイベントが発生しているか（GTM設定の確認に使う） |
| ページ別ユーザー数 | 今どのページを見ているユーザーが多いか |

### 使い方のコツ

- GTMのプレビューでタグを発火させた後、リアルタイムレポートを開いて**自分のアクションが反映されているか確認**する
- 「ユーザースナップショット」で自分の行動ログを1件ずつ追うことができる

> **GTMプレビュー中もGA4に届く**ことに注意。リアルタイムで確認できますが、このデータは通常レポートにも蓄積されます。

---

## 2. イベントレポート

**使い道：** 一定期間に発生したイベントの回数・ユーザー数を集計して確認する。

**場所：** GA4 → レポート → エンゲージメント → イベント

### 見るべき箇所

| カラム | 説明 |
|--------|------|
| イベント名 | `page_view`, `add_to_cart` など |
| イベント数 | 発生した合計回数 |
| ユーザー数 | そのイベントを発生させたユニークユーザー数 |

### ハンズオン後の確認

課題で設定したイベントがここに表示されているか確認しましょう。
（リアルタイムと違い、通常レポートへの反映まで数時間かかることがあります）

---

## 3. コンバージョンレポート

**使い道：** CV数・CV率を確認する。広告代理店では最も重要なレポートのひとつ。

**場所：** GA4 → レポート → エンゲージメント → コンバージョン

### コンバージョンの登録方法

1. GA4 → 管理（左下の歯車）→ イベント
2. 計測されているイベント一覧から対象のイベント名を見つける
3. 右端の「コンバージョンとしてマーク」トグルをONにする
4. しばらくするとコンバージョンレポートに反映される

> ⚠️ **重要：** コンバージョンに登録する前のデータは遡及されません。新しくCVを定義したら、すぐに登録する習慣をつけましょう。

### ハンズオン後の確認

- 課題2で設定した `purchase` イベント
- 課題6-Aで設定した `generate_lead` イベント（お問い合わせ）
- 課題7で設定した `generate_lead` イベント（LP資料請求）

これらをコンバージョンに登録して、コンバージョンレポートに表示されるか確認しましょう。

---

## 4. DebugView（GTM設定確認の最強ツール）

**使い道：** GTMプレビュー中にイベントのパラメータを1件ずつリアルタイムで確認する。リアルタイムレポートより詳細な情報が見られる。

**場所：** GA4 → 管理（左下の歯車）→ DebugView

### 見るべき箇所

| 項目 | 説明 |
|------|------|
| イベントのタイムライン | 送信されたイベントが時系列で表示される |
| パラメータ一覧 | イベント名をクリックすると `product_name`・`company_size` などの値が確認できる |
| デバイス選択 | 複数タブを開いている場合は自分のデバイスを選ぶ |

### 使い方のコツ

- GTMプレビューモードでタグを発火させた直後にDebugViewを確認すると、パラメータの中身をその場で検証できる
- リアルタイムレポートではパラメータの値まで見えないため、**dataLayer変数の確認は必ずDebugViewで行う**
- 課題3の `product_name`、課題7の `company_size` はここで値を確認しよう

> **注意：** DebugViewはGTMプレビュー中（またはデバイスのDebugモードが有効なとき）のみ表示されます。通常の閲覧ではデータが流れません。

## この先のステップ（参考）

研修の範囲を超えますが、実務では以下も重要です：

| ステップ | 内容 |
|---------|------|
| **Google広告との連携** | GA4コンバージョンをGoogle広告にインポートし、広告の自動入札の最適化シグナルとして使う。広告代理店では必須の作業。 |
| **UTMの運用** | 広告URLに `?utm_source=google&utm_medium=cpc` を付与して流入元を正確に計測する |
| **オーディエンス設定** | 特定の行動をとったユーザーをリスト化してリターゲティング広告に活用 |
| **探索レポート** | ファネル・コホート分析など高度な分析（GA4の探索タブ） |
